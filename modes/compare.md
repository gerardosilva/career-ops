# Mode: compare — Multi-Opportunity Comparison

Use a weighted 10-dimension scoring matrix:

| Dimension | Weight | 1-5 Criteria |
|-----------|--------|--------------|
| North Star alignment | 25% | 5 = exact target role, 1 = unrelated |
| CV match | 15% | 5 = 90%+ fit, 1 = <40% fit |
| Level fit | 15% | 5 = staff+, 4 = senior, 3 = mid-senior, 2 = mid, 1 = junior |
| Comp fit | 10% | 5 = top quartile, 1 = below floor |
| Growth path | 10% | 5 = clear next step, 1 = dead end |
| Remote quality | 5% | 5 = strong remote fit, 1 = on-site only |
| Company quality | 5% | 5 = strong signal, 1 = red flags |
| Stack fit | 5% | 5 = very strong Drupal fit, 1 = weak fit |
| Speed to close | 5% | 5 = fast process, 1 = very slow |
| Culture signals | 5% | 5 = strong builder culture, 1 = bureaucratic |

For each opportunity, score every dimension and produce a weighted total.

End with:

- ranked list
- recommendation
- notes on time-to-close and economic fit

Ask the user for the opportunities if they are not already in context.
