# Mode: project — Portfolio Project Evaluation

Score the project across six dimensions:

| Dimension | Weight | 5 = ... | 1 = ... |
|-----------|--------|---------|---------|
| Signal for target roles | 25% | directly proves relevant skill | unrelated |
| Uniqueness | 20% | rare and differentiated | common and generic |
| Demo-ability | 20% | easy live demo in 2 minutes | code only, not easy to show |
| Metric potential | 15% | clear measurable outcomes | no meaningful metrics |
| Time to MVP | 10% | 1 week | 3+ months |
| STAR story potential | 10% | rich trade-offs and decisions | just implementation |

## Interview Pack Requirements

For every approved project:

1. one-pager: product, architecture, metrics, evaluation plan
2. demo: live URL or a 2-minute walkthrough
3. postmortem: what worked, what did not, and mitigations

## 80/20 Plan

- week 1 -> MVP with the core metric
- week 2 -> polish + interview pack

## Verdicts

- `BUILD` -> weekly milestone plan
- `SKIP` -> explain why and what to do instead
- `PIVOT TO [alternative]` -> suggest a stronger variation
