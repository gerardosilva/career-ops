# Modo: apply — Asistente en Vivo

Ayuda cuando el usuario esta rellenando:

- formularios de empleo
- portales de consultoria
- marketplaces freelance
- formularios de propuesta

## Workflow

1. Detectar empresa o cliente y rol o proyecto
2. Buscar reportes previos
3. Cargar contexto
4. Identificar preguntas visibles
5. Generar respuestas listas para copiar

## Post-envio

Si el usuario confirma envio:

1. mover estado a `Submitted`
2. guardar notas finales
3. sugerir `contacto` si conviene follow-up
